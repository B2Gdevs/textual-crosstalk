# Gilgamesh of Uruk

Default soul for this project.

- Narrative: default setup / framework / cross-cutting
- Soul body: `narrative/souls/gilgamesh.md`
- Override path: replace this file with another stable soul pointer when the
  project needs a different voice or decision stance

The active soul is resolved from `SOUL.md` at session start. Use
`gad souls init`, `gad souls add`, and `gad souls use` to manage this pointer.
